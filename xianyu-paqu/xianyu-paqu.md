# 闲鱼爬取商品处理

## 功能说明

从 `E:\Desktop\大灿的知识库\大灿的知识库\raw` 文件夹扫描新的闲鱼 md 文件，处理后在 `E:\Desktop\project\闲鱼商品\` 创建商品文件夹，并维护商品清单。

## 路径配置

| 用途 | 路径 |
|------|------|
| Raw 文件夹 | `E:\Desktop\大灿的知识库\大灿的知识库\raw\` |
| 商品目录 | `E:\Desktop\project\闲鱼商品\` |
| 商品清单 | `E:\Desktop\project\闲鱼商品\商品清单.md` |

## 执行流程

### Step 1: 扫描新文件

1. 扫描 `E:\Desktop\大灿的知识库\大灿的知识库\raw\` 下所有 `*闲鱼.md` 文件
2. 读取商品清单 `商品清单.md`，提取已处理的来源文件列表
3. 对比找出新增文件（不在清单中的）

### Step 2: 处理每个新商品

对每个新 md 文件：

1. **读取内容**：读取完整 md 文件
2. **截取有效内容**：找到 "为你推荐" 文字，保留其之前的全部内容（不含 "为你推荐" 及之后）
3. **提取信息**：
   - 标题：取 md 文件名（去掉 `_闲鱼.md` 后缀）
   - 图片：从 `![](URL)` 格式提取所有图片 URL（商品图片在文件前部，"为你推荐" 之前）
   - 价格：提取 `¥` 后面的数字
   - 描述：保留 "为你推荐" 之前的正文，只做合理分段
4. **创建文件夹**：
   - 文件名：商品标题（20字内，超长截断）
   - 路径：`E:\Desktop\project\闲鱼商品\{商品标题}\`
5. **下载图片**：
   - 下载所有商品图片到 `images/` 子目录
   - 文件名用序号（1.jpg, 2.jpg, 3.jpg...）
   - **注意**：如果下载的文件是 webp 格式（URL含 .webp 或 file 命令显示 Web/P image），需要用 Python PIL 转换为 JPEG：
     ```python
     from PIL import Image
     import os
     for f in os.listdir('images'):
         if not f.lower().endswith(('.jpg', '.jpeg', '.png')):
             img = Image.open(f'images/{f}').convert('RGB')
             new_name = f.rsplit('.', 1)[0] + '.jpg'
             img.save(f'images/{new_name}', 'JPEG')
             os.remove(f'images/{f}')
     ```
6. **写入商品.md**：
   ```markdown
   # {商品标题}

   ## 商品信息
   - 价格：¥{价格}
   - （其他商品信息）

   ## 商品描述
   {分段后的描述正文}
   ```
   - 只做分段整理，不修改原文内容

### Step 3: 更新商品清单

读取现有 `商品清单.md`，追加新商品行：

| # | 商品名称 | 价格 | 文件夹 | 图片数 | 来源文件 |
|---|----------|------|--------|--------|----------|

更新统计部分（商品数量、最后更新时间）。

### Step 4: 如果是增量更新

如果是定时/定期调用（检测到没有新商品），只输出：
```
暂无新增商品
商品清单已存在，共 N 个商品
```

## 分段规则

原文只做合理分段，不修改内容：
- 按主题/功能点拆分成多个自然段
- 保留原文所有信息和措辞
- 合理添加 `##` 二级标题区分"商品信息"和"商品描述"

## 注意事项

- 图片只下载 "为你推荐" 之前出现在商品介绍部分的
- "为你推荐" 之后的内容一律不处理
- 文件夹名控制在 20 字以内
- 商品名称取自 md 文件名（去掉 `_闲鱼.md` 后缀）
